This repository contains multiple ASP.NET Core MVC examples created using C#, HTML, CSS etc.
Each project demonstrates a key feature of the MVC pattern and how it integrates with Entity Framework Core (Code-First) using SQL Server.

# Modules Included

# 1.Entity Relationship (ER) Model Example

Implements Code-First entity relationships: One-to-One, One-to-Many, and Many-to-Many.

Demonstrates model creation, DbContext setup, and migrations with SQL Server.

# 2.Dependency Injection (DI) Example

Explains how to register and inject services in ASP.NET Core using built-in DI container.

Demonstrates loose coupling between controller and service layers.

# 3.Student CRUD Example

Full Create, Read, Update, Delete functionality for managing student data.

Uses Entity Framework Core (Code-First) approach.

Includes form handling, list display, and update/delete confirmation logic.

# 4.Validation Example

Implements server-side validation using Data Annotations ([Required], [EmailAddress], [Range], etc.).

Demonstrates client-side validation using JavaScript for instant feedback.

Covers custom validation attributes and error message display.

# 5.File Upload Example

Shows how to upload and save files to the wwwroot/uploads directory.

Supports file type and size validation.

Implements asynchronous upload method with progress messages.
